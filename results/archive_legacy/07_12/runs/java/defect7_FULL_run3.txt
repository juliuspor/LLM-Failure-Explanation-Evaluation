### Problem Analysis

#### WHAT Happened
A `NullPointerException` is thrown when executing the test: 

```java
assertTrue(Arrays.equals(new Class[] { String.class, null, Double.class }, ClassUtils.toClass(new Object[] { "Test", null, 99d })));
```

#### WHY It Happened
The method `ClassUtils.toClass(Object[] array)` iterates over the given array and attempts to call `.getClass()` on each element. However, if an element in the array is `null`, `.getClass()` will trigger a `NullPointerException`.

#### WHERE It Occurred
In the `toClass` method of `ClassUtils`:

```java
public static Class<?>[] toClass(Object[] array) {
    if (array == null) {
        return null;
    } else if (array.length == 0) {
        return ArrayUtils.EMPTY_CLASS_ARRAY;
    }
    Class<?>[] classes = new Class[array.length];
    for (int i = 0; i < array.length; i++) {
        classes[i] = array[i].getClass();  // Problem line: Will throw exception if array[i] is null
    }
    return classes;
}
```

### Fix Instructions

#### Solution:

1. **Update the Iteration Logic**: Check if an element is `null` before calling `.getClass()`.

2. **Fill Null Values**: If an element is `null`, insert `null` into the resultant `Class<?>[]`.

#### Implementation Steps:

- Modify the `toClass` method as follows:

```java
public static Class<?>[] toClass(Object[] array) {
    if (array == null) {
        return null;
    } else if (array.length == 0) {
        return ArrayUtils.EMPTY_CLASS_ARRAY;
    }
    Class<?>[] classes = new Class[array.length];
    for (int i = 0; i < array.length; i++) {
        if (array[i] != null) {
            classes[i] = array[i].getClass();
        } else {
            classes[i] = null; // Handle null elements explicitly
        }
    }
    return classes;
}
```

### Benefits

- **Prevents NullPointerException**: By handling `null` elements appropriately.
- **Ensures Correct Output**: Return an array with `null` placeholders where needed, matching expected outcomes.
- **Improves Robustness**: Handles a wider range of inputs without error.
