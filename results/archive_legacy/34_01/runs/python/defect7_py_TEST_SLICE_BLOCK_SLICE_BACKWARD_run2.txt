What happened (where it failed)
- The loop at line 765 iterates every index of array.  
- At line 766 the code calls array[i].upper(). That call runs for each element, including values that are not strings.  
- When i == 1 the element is None. On line 766 Python tries to call None.upper(), which raises AttributeError: 'NoneType' object has no attribute 'upper'. The exception stops the loop before classes.append(type(array[i])) at line 767 can run for that element (and before it reaches the float at index 2).

Why it failed (root cause)
- The code assumes every element in array is a string and calls the string method upper() unconditionally. None (type NoneType) and 99.0 (type float) do not have an upper() method, so calling upper() on them raises an AttributeError. The failing call is specifically at line 766: _ = array[i].upper().
