What happened (where and what failed)
- The code checks minutes_offset at lines 396–397:
  if minutes_offset < 0 or minutes_offset > 59:
      raise ValueError(f"Minutes out of range: {minutes_offset}")
  That check threw ValueError: Minutes out of range: -15. So minutes_offset was -15 at the time of that check.
- The check runs before the code that combines hours and minutes (the try block). The combination logic later (lines in the try block) assumes minutes_offset is a non‑negative magnitude to be added or subtracted from hours_in_minutes. But because the guard already rejects negative minute values, an intended negative offset like hours_offset = -2, minutes_offset = 15 (or some callers passing minutes_offset as -15 to indicate negative minutes) will be rejected early.

Why it failed (root cause)
- The function treats minutes_offset inconsistently:
  - It enforces minutes_offset ∈ [0,59] at lines 396–397.
  - Later it mutates minutes_offset to represent the total offset in minutes by subtracting or adding: 
      if hours_in_minutes < 0:
          minutes_offset = hours_in_minutes - minutes_offset
      else:
          minutes_offset = hours_in_minutes + minutes_offset
  - That later code assumes minutes_offset was a non‑negative magnitude (0..59) and that the sign of the total offset comes only from hours_offset. But a caller passed a negative minutes_offset (-15) (or the code expected to accept negative minutes when hours were negative), so the early validation rejected it and threw the ValueError at the validation lines.
- In short: the validation runs at the wrong time (and enforces the wrong shape of minutes_offset for how the function later uses it). The function either expects minutes_offset always non‑negative and combines using hours_offset sign, or expects minutes_offset may already be signed. Currently it demands non‑negative but callers supplied a negative minute value — causing the error.

- Compute total_minutes without reusing minutes_offset for two meanings. Use a separate variable total_minutes:
