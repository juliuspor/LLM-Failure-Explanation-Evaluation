What happened (where it failed)
- Failure occurs inside your loop at line 766: `_ = array[i].upper()`.
- The exception says `'NoneType' object has no attribute 'upper'`. That means array[i] was None for at least one i, so Python tried to call `.upper()` on None and raised an AttributeError.

Why it happened (root cause)
- You check `if array is None:` at the start, which only handles the case where the whole list is None.
- You do not check individual elements inside the list. The list can contain None values (type Optional[List[Any]] allows that).
- When you reach an element that is None, the call `array[i].upper()` tries to call upper on a None object and fails.

Where exactly (line references)
- Line 765: `for i in range(len(array)):` — iterates elements.
- Line 766: `_ = array[i].upper()` — raises the AttributeError when array[i] is None.
