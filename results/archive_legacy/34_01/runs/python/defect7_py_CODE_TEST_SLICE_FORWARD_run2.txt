Example corrected implementation (simple and clear)
    def to_class(cls, array: Optional[List[Any]]) -> Optional[List[type]]:
        if array is None:
            return None
        elif len(array) == 0:
            return ArrayUtils.EMPTY_CLASS_ARRAY.copy()
        classes = []
        for i in range(len(array)):
            # Only call .upper() for strings
            if isinstance(array[i], str):
                _ = array[i].upper()
            classes.append(type(array[i]))
        return classes

This change prevents calling .upper() on None and lets the test pass: to_class(['Test', None, 99.0]) -> [str, type(None), float].
