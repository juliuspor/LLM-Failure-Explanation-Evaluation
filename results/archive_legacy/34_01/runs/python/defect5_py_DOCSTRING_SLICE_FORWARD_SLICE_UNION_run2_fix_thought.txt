The bug is that when array is None and element is not None, inferred_type was set to object instead of type(element), leading to spurious TypeError when expected_type provided. Fix by inferring the actual element type when element is present, and change type check to validate the element against expected_type: if expected_type provided and element is None but expected_type != object -> raise; if element not None and not isinstance(element, expected_type) -> raise. Keep using _copy_list_grow1 to build the new list and assign element to the last position.
