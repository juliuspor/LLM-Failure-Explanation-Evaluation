- That makes the return either (a) improperly unindented so the parser raises "expected an indented block" or (b) if the return kept its indentation level but is no longer attached to the def, a "return outside function" / syntax error occurs.
- Concretely, removing the docstring line (line 450) broke the contiguous, indented body of get_max_middle_index, so the interpreter fails to parse the function.
