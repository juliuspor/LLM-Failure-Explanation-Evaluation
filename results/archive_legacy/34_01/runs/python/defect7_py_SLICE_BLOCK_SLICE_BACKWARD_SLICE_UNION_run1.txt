- The loop assumes every `array[i]` is a string. At least one element in `array` is a different type (for example an int, None, or a custom object). Calling `.upper()` on that element fails because that type does not implement `.upper()`.

- Index: `i`
- Failing call: `array[i].upper()` on line 766
