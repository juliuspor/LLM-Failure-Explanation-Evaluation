Why it happened (root cause)
- The code unconditionally calls array[i].upper() on every element (variable array[i]) without checking its type or whether it is None.
- None has no method upper(), so array[i].upper() fails as soon as an element is None.
- The line with upper() appears unnecessary for the goal of this function (you only need the element's class/type), and it contradicts the docstring/test intent to accept None elements.
