add_number previously used self.add which ultimately called self.append leading to AttributeError because append is not defined. To avoid calling append, this replacement will call maybe_end_statement to preserve statement-ending behavior, compute previous char using get_last_char, and then directly concatenate to self.code the necessary pieces (spaces, numbers, scientific notation) without using append/add. It preserves original numeric formatting logic and handles negative numbers and integer formatting with exponent optimization.
