What happened (where it failed)
- The error occurred inside your method to_class, on the loop at the lines shown in the trace:
  - line 765: for i in range(len(array)):

- The Python exception is AttributeError: 'NoneType' object has no attribute 'upper'. That means array[i] was None on at least one iteration, so calling array[i].upper() fails because None has no method upper().

Why it happened (root cause)
- Your code assumes every element in array is a string (has .upper()). That assumption is false for at least one element: array contains None.
- The code accesses array[i] directly and immediately calls .upper() without any None check or type check. When array[i] == None, Python raises AttributeError at line 766.
