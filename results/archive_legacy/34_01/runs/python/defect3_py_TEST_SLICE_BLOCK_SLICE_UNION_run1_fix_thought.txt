Build a mapping from Python string codepoint indices to UTF-16 code unit indices while creating code_units. Iterate over Python character indices (codepoint indices), and for each, translate by calling translate_codepoint with the corresponding code_units index. When a translator consumes N codepoints, advance the codepoint index by N. When no translation occurs, append the original codepoint's UTF-16 units to out and advance by 1.
