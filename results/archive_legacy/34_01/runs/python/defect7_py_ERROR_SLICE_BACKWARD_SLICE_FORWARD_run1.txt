What happened (where it failed)
- The exception happened at line 766 when this code ran:
  _ = array[i].upper()
- Python raised AttributeError: 'NoneType' object has no attribute 'upper' because array[i] was None at that index.

Why it happened (root cause)
- The call array[i].upper() assumes array[i] is a string (or another object that implements upper()). At runtime, one element in array is the value None, which does not have an upper() method. So the interpreter stops with that AttributeError.

1. Add a quick check before line 766 to see which index is None:

       print("Found None at index", i)
2. Or inspect the array beforehand:

       if val is None:
           print("None at", idx)
