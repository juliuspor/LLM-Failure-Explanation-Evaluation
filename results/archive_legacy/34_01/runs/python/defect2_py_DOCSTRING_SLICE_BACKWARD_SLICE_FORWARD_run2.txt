   - At line 71 you compute

   - If self._upper_bound == self._lower_bound, the denominator is zero and Python raises ZeroDivisionError before any further code runs.

2. ValueError raised at line 75 when the normalized gray value is out of range:
   - If the denominator is non‑zero but value is outside the expected range, g can be < 0 or > 255. The code intentionally raises a ValueError at line 75:
