What happened (where it failed)
- At line 766 the code does: array[i].upper()
- With the input array = ['Test', None, 99.0], when i == 1 array[1] is None, so array[1].upper() raises AttributeError: 'NoneType' object has no attribute 'upper'.
- Even if the None were removed, when i == 2 array[2] is 99.0 (a float) and 99.0.upper() would raise the same error because float has no upper() method.

Why it failed (root cause)
- The loop assumes every element of array is a string and directly calls the string method upper() without checking the element’s type.
- That assumption is false for the provided input; elements can be None or non-strings, so calling upper() is invalid.

Where exactly in the code
- The problem occurs inside the loop shown:
  - line 765: for i in range(len(array)):
  - line 766:     _ = array[i].upper()
- The exception will be raised on the first non-string element (here at i == 1).

Concrete fixes (step-by-step options)
1) Only call upper on strings (recommended if you only want to work on string values)
   - Replace the body of the loop with a type check:
     for i in range(len(array)):
         if isinstance(array[i], str):
             _ = array[i].upper()
         else:
             _ = array[i]   # or handle non-strings explicitly

2) Convert values to strings before calling upper (if you want to treat None/99.0 as strings)
   - Use str() safely:
     for i in range(len(array)):
         _ = str(array[i]).upper()

3) Use try/except to handle unexpected types
   - Catch the AttributeError and decide how to handle it:
     for i in range(len(array)):
         try:
             _ = array[i].upper()
         except AttributeError:
             _ = array[i]  # or convert/do fallback

4) Use enumerate for clearer code (and combine with one of the above)
   - Example with isinstance:
     for i, val in enumerate(array):
         if isinstance(val, str):
             processed = val.upper()
         else:
             processed = val
         # store processed back if needed: array[i] = processed

Choose the option that matches your intended behavior: skip non-strings, convert everything to strings, or raise a clearer error.
