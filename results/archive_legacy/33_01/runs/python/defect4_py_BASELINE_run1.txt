What happened (root cause)
- In the "max middle" section you fetch the wrong existing period. Instead of using self._max_middle_index you call get_data_item(self._min_middle_index):
  if self._max_middle_index >= 0:
      max_middle_period = self.get_data_item(self._min_middle_index).get_period()
                                                          ^^^^^^^^^^^^^^^^^^^^^^
- Because you read the period at _min_middle_index, you compute max_middle from the wrong period and therefore compare "middle" against the wrong value. That can cause _max_middle_index to be updated incorrectly (in your failing test it becomes 3 instead of the expected 1).

Where it occurs
- The bug is in the block that starts with:
  if self._max_middle_index >= 0:
- Specifically the call to get_data_item uses self._min_middle_index instead of self._max_middle_index.

Why the test fails
- Your test inserts four SimpleTimePeriod items whose middles are:
  index 0: (0+100)//2 = 50
  index 1: (100+200)//2 = 150  <-- expected max
  index 2: (0+50)//2 = 25
  index 3: (50+100)//2 = 75
- The code should compare a new period's middle against the current max (index 1), but because it reads the period at _min_middle_index (probably 2 or 0 depending on previous updates) it compares against the wrong middle and ends up setting _max_middle_index to 3.

Exact fix (what to change)
Replace the incorrect line in the max-middle block:
- Change:
    max_middle_period = self.get_data_item(self._min_middle_index).get_period()
- To:
    max_middle_period = self.get_data_item(self._max_middle_index).get_period()

Full corrected block:
if self._max_middle_index >= 0:
    max_middle_period = self.get_data_item(self._max_middle_index).get_period()
    if isinstance(max_middle_period, SimpleTimePeriod):
        s = max_middle_period.get_start_millis()
        e = max_middle_period.get_end_millis()
    else:
        s = int(max_middle_period.get_start().timestamp() * 1000)
        e = int(max_middle_period.get_end().timestamp() * 1000)
    max_middle = s + (e - s) // 2
    if middle > max_middle:
        self._max_middle_index = index
else:
    self._max_middle_index = index
