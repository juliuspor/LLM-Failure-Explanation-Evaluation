Example corrected get_paint implementation

def get_paint(self, value):
    # clamp input to [0.0, 1.0]
    v = max(0.0, min(1.0, value))

    return (v, v, v)

Why that fix works
- Clamping prevents negative or >1 values from producing out-of-range channel numbers.
- Converting to the appropriate numeric type (int vs float) matches what the color API expects, avoiding validation errors.
