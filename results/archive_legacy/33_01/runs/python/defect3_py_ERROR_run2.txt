What the message means
- "IndexError: String index out of range: 2" means your code attempted to access character position 2 of a string that does not have a character at index 2.
- Indexing is zero-based in most languages: index 0 is the first character, index 1 is the second, index 2 is the third. If the string length is less than or equal to the index you used, you get this error.
