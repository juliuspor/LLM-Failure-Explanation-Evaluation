2) Wrong comparison operator (>= vs >)

     This overwrites an earlier index when a later element has equal score. If index 1 had the intended best_score and index 3 has the same score, >= will pick 3; > would preserve the first occurrence (index 1).

3) Mutated or incorrectly sliced input

4) Integer division / rounding differences (if score depends on division)
