What happened
- The exception occurs in ClassUtils.to_class, at these lines (lines 768–770 in your slice):
  - _ = array[i].upper()
  - classes.append(type(array[i]))
- The test calls ClassUtils.to_class(['Test', None, 99.0]).
- On the second element array[1] is None. None has no method upper(), so calling array[i].upper() raises AttributeError: 'NoneType' object has no attribute 'upper'.

Why this line exists and why it fails
- The call to .upper() appears intended to validate or touch string elements, but it runs unconditionally for every element in the input array.
- The method contract (docstring and tests) expects None elements to be allowed and mapped to None in the resulting class list. However the .upper() call prevents reaching that behavior because it crashes before type(None) can be appended.
- This happens inside to_class (ClassUtils.to_class), so the failure is local to that method (around lines 768–770).

Where to look in the code
- Method: ClassUtils.to_class (the for-loop that iterates array items).
- Exact offending line: _ = array[i].upper() (line 769 in your slice).

Concrete fixes (step‑by‑step)
1. Remove the unconditional .upper() call (it’s not needed to determine the type).
   - Replace the loop body with something like:
     - if array[i] is None:
         classes.append(None)
       else:
         classes.append(type(array[i]))
2. If the intention was to validate that non-None strings are valid text, first check for None:
   - if array[i] is None:
       classes.append(None)
     elif isinstance(array[i], str):
       _ = array[i].upper()  # only for string validation
       classes.append(str)
     else:
       classes.append(type(array[i]))
3. Add unit tests covering None elements (you already have one) and other types to ensure no AttributeError is raised.

Why these fixes work
- They avoid calling string methods on None, so no AttributeError is raised.
- They keep the method behavior consistent with the docstring/test: a None element produces None in the returned class array.

Minimal patch example

for i in range(len(array)):
