- str.upper() is a method that exists only on string objects. Some element(s) in array are not strings (for example an int, None, list, etc.).
- When Python evaluates array[i].upper() for a non‑string element, that element has no .upper attribute and Python raises an AttributeError immediately. The loop stops at the first non‑string element, so classes.append(type(array[i])) on line 770 never runs for that iteration.
