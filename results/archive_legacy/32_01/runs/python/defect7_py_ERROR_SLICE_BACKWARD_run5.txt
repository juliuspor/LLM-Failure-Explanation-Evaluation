What happened (what the error message means)
- At line 769 your code calls the string method upper(): array[i].upper().
- Python raised AttributeError: 'NoneType' object has no attribute 'upper' because array[i] was None for at least one index.
- NoneType is the type of the value None. None has no .upper() method, so the call fails.

Why it happened (root cause)
- Some element in array is not a string but the special value None. Possible reasons:
  - array was built from mixed inputs and one entry was missing or returned None.
  - a previous operation replaced or failed to produce a string for that index.
  - values came from a source (e.g., CSV column, function return, dict lookup) that sometimes yields None.
- The code assumes every element is a string and calls .upper() without checking.

Where it occurred (exact location)
- The error occurred at these lines in your slice:
 768 |         for i in range(len(array)):
 769 |             _ = array[i].upper()
- The failing call is at line 769, index i inside the loop.

How to fix it (concrete steps and example code)
Option A — Validate and raise a clear error (stop early and show index)
- This is good when None is unexpected and you want to find bad data.
Replace the two lines with:
for i in range(len(array)):
    if array[i] is None:
        raise ValueError(f"array[{i}] is None; expected a string")
    _ = array[i].upper()

Option B — Skip None values (if None means “no value” and should be ignored)
for i in range(len(array)):
    if array[i] is None:
        continue
    _ = array[i].upper()

Option C — Coerce to string (if you want a string regardless)
for i in range(len(array)):
    _ = str(array[i]).upper()

Option D — Work more Pythonically with enumerate and type check
for i, value in enumerate(array):
    if not isinstance(value, str):
        raise TypeError(f"array[{i}] is {type(value).__name__}; expected str")
    _ = value.upper()

- Trace back to where array is created or populated and ensure functions or data sources either return strings or handle missing values.
