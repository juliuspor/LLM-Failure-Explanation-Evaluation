- array contains at least one element that is not a string (for example an int, None, bytes, or some other object).
- Calling .upper() only works on string objects. When a non-string item is hit, Python raises AttributeError because that object has no upper() method.
- The next line (770) is not reached for that iteration because the exception stops the loop at line 769.
