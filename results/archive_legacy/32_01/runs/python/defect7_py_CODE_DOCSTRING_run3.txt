What happened (where it fails)
- The failure occurs inside to_class, in the for loop on the line:
  _ = array[i].upper()
  This attempts to call .upper() on each element of array.

Why it fails (root cause)
- If any element is None, calling array[i].upper() raises AttributeError: 'NoneType' object has no attribute 'upper'. That exactly matches the observed exception.
- Even if elements are not None, many non-string objects (ints, lists, custom objects) do not implement .upper(), which will raise AttributeError too.
- The .upper() call is unnecessary for the stated purpose ("Converts an array of Object to an array of Class objects"). The code already uses type(array[i]) to get the class — .upper() serves no valid purpose here and appears to be a leftover or mistaken line.

Where it occurs (specific method/line)
- Method: to_class
- Problematic line: the call to array[i].upper() inside the for loop (the line that assigns to the unused variable _).

Why the docstring and code disagree
- The docstring claims that if an element is None, a None element should be inserted into the result. The current code does the opposite: it will raise AttributeError as soon as it encounters None (or any element without .upper()).

Concrete fixes (step-by-step)
1. Remove the extraneous .upper() call.
2. Handle None elements explicitly: append None to classes when array[i] is None.
3. Optionally, keep type checking for other elements by appending type(array[i]).
4. Review ArrayUtils.EMPTY_CLASS_ARRAY.copy() — ensure ArrayUtils is defined and that copy() returns the expected list (or replace with [] if not needed).

Minimal corrected implementation
    def to_class(cls, array: Optional[List[Any]]) -> Optional[List[Type]]:
        if array is None:
            return None
        elif len(array) == 0:
            return ArrayUtils.EMPTY_CLASS_ARRAY.copy()
    
        classes = []
        for i in range(len(array)):
            if array[i] is None:
                classes.append(None)
            else:
                classes.append(type(array[i]))
        return classes
