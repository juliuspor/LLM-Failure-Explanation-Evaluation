What happened (where it failed)
- The failing call is ClassUtils.to_class(['Test', None, 99.0]).
- Inside to_class, the loop at lines 768–770 runs for each element:
  - Line 769 calls array[i].upper()
  - Line 770 appends type(array[i]) to classes
- When i reaches the second element (None), array[i].upper() raises AttributeError because NoneType has no .upper() method. The code never reaches line 770 for that element and the test fails.

Why it failed (root cause)
- The code assumes every array element implements .upper() (i.e., is a string) and calls it unconditionally. That assumption is false: None and floats (99.0) do not have .upper().
- The unnecessary call to .upper() is the direct cause. The goal of to_class (based on the test) is to collect types, so calling .upper() is both unnecessary and unsafe.

Exactly where (reference)
- Method: to_class
- Problem lines: 768–770
  - 768: for i in range(len(array)):
  - 769:     _ = array[i].upper()    <- raises AttributeError on None (and on other non-str values)
  - 770:     classes.append(type(array[i]))

Concrete fixes (step-by-step, pick one)

1) Remove the .upper() call (recommended)
- Replace the loop body with just:
  - classes.append(type(array[i]))
- Why: to_class only needs the element type; calling .upper() is unnecessary and unsafe.

2) Guard the .upper() call with a type check
- If .upper() is needed for some reason, only call it for strings:
  - if isinstance(array[i], str):
      _ = array[i].upper()
    classes.append(type(array[i]))
- This prevents AttributeError for None and other non-strings.

3) Use try/except around the call
- Catch AttributeError and proceed:
  - try:
      _ = array[i].upper()
    except AttributeError:
      pass
    classes.append(type(array[i]))
- This is less explicit than (2) and hides other possible mistakes, so use it only if you expect many different object types and calling .upper() is optional.

Example minimal patch (preferred)
- Replace lines 768–770 with:
  768 |         for i in range(len(array)):
  769 |             classes.append(type(array[i]))

That change makes the test pass: to_class(['Test', None, 99.0]) -> [str, type(None), float].
