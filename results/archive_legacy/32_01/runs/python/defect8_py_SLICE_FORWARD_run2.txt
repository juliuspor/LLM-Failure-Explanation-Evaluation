  Because that return executes every time control reaches line 248, the later code at lines 250–252 can never run. That makes the variant-handling and the sanity check at line 250 (if locale_str[5] != '_': raise ValueError(...)) dead code.

- In short: the unconditional return at line 248 prevents both validation and the variant-return path from ever executing.
