The error message, `AttributeError: 'NoneType' object has no attribute 'upper'`, indicates that you're trying to call the `upper()` method on a `None` value. This likely occurs in a section of code where the method assumes that all elements in a list are non-null strings or other objects with an `upper()` method.

Looking at the code, the problem occurs in the `to_class` method of the `ClassUtils` class:

```python
def to_class(cls, array: Optional[List[Any]]) -> Optional[List[Type]]:
    if array is None:
        return None
    elif len(array) == 0:
        return ArrayUtils.EMPTY_CLASS_ARRAY.copy()

    classes = []
    for i in range(len(array)):
        _ = array[i].upper()  # Problematic line
        classes.append(type(array[i]))
    return classes
```

In this code, the for-loop iterates over each element in the `array` list, and there is an immediate attempt to call the `upper()` method on each element with `array[i].upper()`. The code does not check if the element is `None`, which causes the error when `None` is encountered in the list.

### What Happened:
- The function `to_class()` expects to convert objects to their types, but a line of code (`array[i].upper()`) assumes that each object in the list is a string.
- If any element in the input list `array` is `None`, this call results in an `AttributeError` because `NoneType` objects have no `upper()` method.

### Why It Happened:
- The absence of a check for `None` leads to an error when the list contains `None` values.

### Where It Occurred:
- Inside the `to_class()` method, specifically at this line: `array[i].upper()`.

### How to Fix It:
To fix the error, check if an element is `None` before attempting to call `upper()` on it. The `upper()` call is not needed to determine the type, so you should remove that line if class types are all that's required.

Here's a corrected snippet that avoids the error:

```python
def to_class(cls, array: Optional[List[Any]]) -> Optional[List[Type]]:
    if array is None:
        return None
    elif len(array) == 0:
        return ArrayUtils.EMPTY_CLASS_ARRAY.copy()

    classes = []
    for item in array:
        if item is not None:
            classes.append(type(item))
        else:
            classes.append(None)
    return classes
```

This modification includes:
- A check to ensure that `item` is not `None` before using it.
- Simply appending `None` to the `classes` list when an item is `None`.
