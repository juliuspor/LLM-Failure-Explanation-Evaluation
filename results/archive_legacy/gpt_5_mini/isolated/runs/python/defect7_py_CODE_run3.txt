Short answer: the code crashes because of this line:
    _ = array[i].upper()
That line assumes every element in array is a string. If any element is not a string (for example an int or None) Python raises AttributeError: 'int' object has no attribute 'upper' (or 'NoneType' object has no attribute 'upper'), and execution stops before the function can build classes.
