What happened (where it failed)
- At line 766 the code does: array[i].upper()
- If array[i] is None this causes Python to raise AttributeError: 'NoneType' object has no attribute 'upper'
- The exception originates on line 766 inside the loop that starts on line 765

Why it failed (root cause)
- The code assumes every element in array has an .upper() method (i.e. is a string-like object).
- The docstring says "If any of these objects is None, a None element will be inserted into the array" — but the implementation calls .upper() unconditionally, so None elements are not handled before method calls.
- Because None does not implement .upper(), Python raises AttributeError when the code attempts array[i].upper().
