Why it failed (root cause, with names and line numbers)
- Method: to_class (first line of the snippet).
- At line 766 you do: _ = array[i].upper().
- At line 767 you then use: classes.append(type(array[i])).
- The presence of type(array[i]) on line 767 indicates the elements in array are already class/type objects, not strings.
- Calling .upper() only makes sense on a str. When array[i] is a class (a type object), array[i].upper triggers AttributeError because type objects do not have an upper() method.
- The exception therefore occurs at line 766 and stops the loop.
