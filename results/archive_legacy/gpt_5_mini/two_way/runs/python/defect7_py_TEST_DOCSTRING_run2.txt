A minimal correct implementation (copy/paste-ready)

def to_class(array):
    if array is None:
        return None
    result = []
    for element in array:
        if element is None:
            result.append(type(None))
        else:
            result.append(type(element))
    return result

Why this fixes it
- It never calls a method or accesses attributes on None, so no AttributeError.
- It returns the class objects directly (type(obj)), matching the tests' expectations.
- It handles None input (returns None) and empty lists correctly.
