- The code then enters the loop at line 181:

   - The first evaluation of check_val uses exp. That will raise a NameError at line 182 because exp has never been assigned. The code never gets past that statement.

2) If exp is defined earlier

Using exp inside the loop without initialization causes a NameError. If exp happens to have a value, the loop becomes infinite because nothing changes to make mantissa % 10 != 0 or to exit.

Where it occurs
