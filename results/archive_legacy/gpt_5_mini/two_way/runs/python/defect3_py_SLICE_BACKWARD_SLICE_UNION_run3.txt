  def to_utf16_code_units(s: str) -> List[int]:
      # populate code_units from s
      b = s.encode('utf-16-le')            # bytes: 2 bytes per code unit
      code_units = [int.from_bytes(b[i:i+2], 'little') for i in range(0, len(b), 2)]
      return code_units

- After this change, the later decoding/indexing logic will have a non-empty list to work with.

2) Populate code_units by computing code units from code points (explicit surrogate handling)
- After line 12, iterate over each code point ordinal (ord(c)) and:
  - If ord <= 0xFFFF, append it.
  - If ord > 0xFFFF, compute surrogate pair and append the two 16‑bit values.
- Example implementation:

  def to_utf16_code_units(s: str) -> List[int]:
      code_units = []
      for ch in s:
          cp = ord(ch)
          if cp <= 0xFFFF:
              code_units.append(cp)
          else:
              cp -= 0x10000
              high = 0xD800 + (cp >> 10)
              low  = 0xDC00 + (cp & 0x3FF)
              code_units.extend([high, low])
      return code_units

- This yields the same list of 16‑bit code units and avoids the IndexError.

- Use option 2 if you want explicit control over surrogate generation and clarity about code unit values.
