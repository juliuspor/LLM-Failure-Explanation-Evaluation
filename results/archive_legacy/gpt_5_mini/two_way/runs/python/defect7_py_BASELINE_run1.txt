2. Append the element's type directly. For None elements, type(None) will be appended (matches your tests).

    def to_class(cls, array: Optional[List[Any]]) -> Optional[List[type]]:
        if array is None:
            return None
        if len(array) == 0:
            return ArrayUtils.EMPTY_CLASS_ARRAY.copy()
        classes = []
        for elem in array:
            classes.append(type(elem))
        return classes

Why this fixes the tests
