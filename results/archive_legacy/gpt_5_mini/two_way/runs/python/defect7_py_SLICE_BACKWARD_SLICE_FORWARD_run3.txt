What happened (where it failed)
- The failure happens at line 766 in the loop:
  766 |             _ = array[i].upper()
- That line calls the string method upper() on array[i]. The error you get is an AttributeError like "'int' object has no attribute 'upper'" (or "'NoneType' object has no attribute 'upper'") because not every element in array is a str.

Why it failed (root cause)
- upper() only exists on str objects. If array contains any non-string value (for example an int, None, bytes, a custom object), calling array[i].upper() triggers an AttributeError.
- The code assumes every item is a string without checking. Because the loop indexes every element (range(len(array))), the first non-string element encountered at some index i causes the exception and stops the loop before line 767 runs.
