What happened (where it failed)
- The test error tells you the exception came from trying to call .upper() on a None value: AttributeError: 'NoneType' object has no attribute 'upper'.
- The slice shows the exact place in your code:

  - line 767: classes.append(type(array[i]))
- So at runtime, array[i] was None for at least one i, and the code attempted to call array[i].upper().

Why it happened (root cause)
- .upper() is a string method. Calling array[i].upper() assumes array[i] is a string (or at least non-None with an upper method).
- One or more elements of array are None. The code does not check for None before calling .upper(), so when array[i] is None Python raises AttributeError.

- Add a diagnostic print or assert before line 766:
