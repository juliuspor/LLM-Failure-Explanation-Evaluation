Why it failed (root cause)
- The code assumes replacement_list[i] (and search_list[i]) are strings when it computes lengths to estimate buffer increase. It does not verify that replacement_list[i] is not None before calling len(), so a None element triggers TypeError.
- In short: there is a missing None check before len(replacement_list[i]) at line 82.

How to fix it (clear steps and exact places to change)
1. Add a None check before any len(...) call that touches elements of search_list or replacement_list. Concretely, modify the loop at lines 81–85 so it only computes lengths for non-None entries.
