What happens
- Python raises an IndentationError (or SyntaxError: unexpected indent) at the line that contains
  mantissa //= 10.
- The problem is not logic: it’s a broken indentation inside the while/if block. In the [SLICE_BLOCK] the line numbers show that line 184 ("mantissa //= 10") is indented differently than the surrounding lines:

  - Line 182: check_val = ...
  - Line 183: if check_val == value:
  - Line 184:  mantissa //= 10    ← has one extra space (unexpected indent)
  - Line 185:   exp += 1

Why this fails (root cause)
- Python uses indentation, not braces, to determine block structure. Every statement in the same block must have the same indentation. The extra leading space on line 184 makes Python see that line as starting a new, more-indented block where none is allowed, so it throws an "unexpected indent" error.
- The error occurs inside the while loop/if block beginning at lines 181–186 (while mantissa % 10 == 0: ...). Variable names involved are mantissa, check_val, value, exp.

Exact place to change
- Fix the indentation of the line at 184 so it matches the indentation level of lines 183 and 185 (the statements that belong to the if check_val == value: block).
- Ensure the four lines in that block have consistent indentation:
  - 182: check_val = ...
  - 183: if check_val == value:
  - 184:     mantissa //= 10
  - 185:     exp += 1
  - 186: else:
  - 187:     break

How to fix (concrete steps)
1. Open the file in a text editor that shows whitespace (or run "sed -n '181,187p' file.py" to view raw lines).
2. On the mantissa //= 10 line (line 184), remove the extra leading space so it uses the same number of spaces as "exp += 1".
3. Make sure the file uses consistent indentation throughout (prefer 4 spaces per indent). If there are mixed tabs/spaces, convert tabs to spaces:
   - In many editors: "Convert tabs to spaces" or reindent selection.
   - Or run: python -m tabnanny . to find inconsistent indentation, or use autopep8/black to reformat.
4. Re-run the code; the IndentationError should be gone.
