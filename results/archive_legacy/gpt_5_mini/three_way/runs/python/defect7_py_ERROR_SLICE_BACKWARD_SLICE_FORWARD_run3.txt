Why this happened (root cause)
- .upper() is a string method. It only exists on str objects.
- Calling .upper() on a value that is None causes Python to try to find .upper on a NoneType, which does not exist — hence the AttributeError.
- The root cause is that the array contains at least one None element, but the code assumes every element is a str and calls .upper() unconditionally.
