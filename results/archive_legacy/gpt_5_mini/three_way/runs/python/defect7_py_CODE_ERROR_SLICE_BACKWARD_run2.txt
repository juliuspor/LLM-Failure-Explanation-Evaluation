What happened (root cause)
- The exception occurs at line 766 where the code calls array[i].upper().
- That call assumes array[i] is a string (has an upper() method). In your failing test array[i] is None, so array[i].upper() raises AttributeError: 'NoneType' object has no attribute 'upper'.
- In short: the code never checks for None (or for non-string items) before calling .upper().

Where it happened
- In your pasted slice:
  - Line 765: for i in range(len(array)):
  - Line 766:     _ = array[i].upper()
- The failing variable is array[i] at line 766.

Why this happened (step-by-step)
1. to_class is called with array that contains at least one None element.
2. The loop iterates to that index and executes array[i].upper().
3. None has no upper attribute, so Python raises AttributeError and the function fails immediately.
