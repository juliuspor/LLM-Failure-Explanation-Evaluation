What happened (where it fails)
- At line 766 you call array[i].upper(). If array[i] is None, None has no method named upper, so Python raises AttributeError on that statement.
- The crash happens inside the loop at line 766 (for i in range(len(array)):), before the code reaches the append on line 767.

Why it fails (root cause)
- The code assumes every element in array is an object that implements .upper(). That assumption is false when an element is None.
- The docstring says "If any of these objects is None, a None element will be inserted into the array" but the code does the opposite: it calls .upper() unconditionally and therefore raises AttributeError instead of inserting None.
