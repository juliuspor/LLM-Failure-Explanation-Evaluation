What happened (root cause)
- Python raised an IndentationError because the three lines inside the inner if/while are not consistently indented.
- Specifically: at slice block lines 182–186 you have
  - 182: check_val = ...
  - 183: if check_val == value:
  - 184:  mantissa //= 10
  - 185:   exp += 1
  - 186: else:
The line 184 (mantissa //= 10) is indented with a different amount of whitespace than line 185 (exp += 1) and the surrounding lines. That mismatch makes the parser fail with an IndentationError (or "unindent does not match any outer indentation level").

Where it occurred
- Function: add_number
- Block: inside the integer-special-case branch starting at line 176
- Exact place: the body of the inner if inside the while loop — slice block lines 181–186 (the mantissa //= 10 and exp += 1 lines).

Why it happened
- Python requires consistent indentation (same number of spaces or tabs) for a given block. One or more of those lines use a different indentation width or mixed tabs/spaces, so the parser cannot determine the block structure and fails.

What to do (concrete fixes, step-by-step)
1. Open the file in an editor that can show invisible characters (or use a command):
   - Command-line: sed -n l file.py or cat -A file.py or expand -t 4 file.py
   - Or open in VS Code / PyCharm and enable "render whitespace".
2. Inspect lines 182–186. You should see that the indentation for "mantissa //= 10" is different from "exp += 1" and from the "if" line.
3. Make all indents consistent for that block. Use 4 spaces per indent level (or whatever your project uses). For example:
       if check_val == value:
           mantissa //= 10
           exp += 1
       else:
           break
4. Ensure you are not mixing tabs and spaces:
   - Convert tabs to spaces (in many editors: "Convert Indentation to Spaces" or run: expand -t 4 infile > outfile).
5. Re-run the code. The IndentationError should be resolved.

Optional checks
- Run python -m pyflakes or flake8 to catch indentation and style issues earlier.
- Configure your editor to insert spaces for tabs and show whitespace.
