What happened
- At line 766 the code calls array[i].upper(). If array[i] is None, Python raises an AttributeError because None has no method upper(). That exception stops the loop before you ever append anything for that element (line 767), so the function fails instead of returning a class list or inserting a None element.

Where it occurs
- The crash happens exactly at line 766:
  766 |     _ = array[i].upper()
 766 is the call that fails when array[i] is None.

Why it fails (root cause)
- The code assumes every element in array is an object that implements upper() (a string). That assumption is false when an element is None. There is no safeguard (no None check) before calling .upper(), so None triggers an AttributeError.

How to fix it (concrete steps you can apply)

   - if array is None: return None

3. Remove the unnecessary .upper() call if you only need the type:
   - The call to array[i].upper() serves no purpose for computing type(array[i]). Delete line 766 entirely (or only use it when you expect strings and need to normalize/validate them).
