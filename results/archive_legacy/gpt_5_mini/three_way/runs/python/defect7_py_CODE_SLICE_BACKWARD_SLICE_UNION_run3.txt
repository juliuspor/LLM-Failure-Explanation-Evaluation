- What happened: an AttributeError is raised because array[i] is not a string and therefore has no .upper() method (for example it might be a class object, None, an int, etc.).
- Why it happens: the code assumes every element of array is a str, but it isn't. Calling .upper() on a non-str value fails immediately.

Why that explains the rest of the method
- Because the method calls .upper() unguarded before doing anything else, the code never reaches classes.append(...). So you never build the list of classes.
