What happened (where it failed)
- The error occurred at line 52: the code does this:

  and Python raised TypeError: object of type 'NoneType' has no len().
- That means search_list is None at the time len(search_list) is called.

Why it happened (root cause)
- The code never checks that the top-level search_list argument itself is not None before using len(search_list).
- Later lines (102–105) do check individual entries:
    or search_list[i] is None
    or len(search_list[i]) == 0
    or replacement_list[i] is None
  but those checks run only after search_length was computed and after the code started iterating. They do not protect the initial len(search_list) call.
- In short: a None value was passed in (or left unset) for search_list, and the function immediately called len(...) on it at line 52, causing the TypeError.
