- The condition at lines 239–240 (ch3 < 'A' or ch3 > 'Z' or ch4 < 'A' or ch4 > 'Z') therefore fails for ch3 == '_' and raises the ValueError.

  - indices: 0='f', 1='r', 2='_', 3='_', 4='P'...
  - Your code expects uppercase letters at indices 3 and 4, but index 3 is '_' so it raises.
