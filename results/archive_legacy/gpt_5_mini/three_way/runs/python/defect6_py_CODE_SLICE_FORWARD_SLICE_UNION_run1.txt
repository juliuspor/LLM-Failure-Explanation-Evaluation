- exp was not initialized in all code paths. The code uses exp both inside the while and after the if-block, so exp must be defined unconditionally before those uses.

   - Insert: exp = 0 on the line after mantissa = value (i.e., add exp = 0 at line 179).
2. Keep the rest of the logic the same. With exp initialized to 0, the while loop and later if exp > 2 will work correctly whether or not the abs(x) >= 100 branch runs.

    mantissa = value

    if abs(x) >= 100:
