  - AttributeError: 'NoneType' object has no attribute 'upper'

Why it happens (root cause)
- The code assumes every element in array has a string-like .upper() method. That assumption is false:
  - None has no .upper().

- The extra call to .upper() is unnecessary for determining the element's class (type). Using .upper() also prevents correct handling of None and other non-string elements.
