We need to make get_paint robust against zero denominator and out-of-range values. The function will: check that upper_bound > lower_bound and raise ValueError if not; clamp the input value to the [lower_bound, upper_bound] range; compute the normalized fraction using the clamped value; compute g as int(round(...)) and clamp to 0..255; return (g,g,g). This avoids ZeroDivisionError and guarantees valid RGB values.
