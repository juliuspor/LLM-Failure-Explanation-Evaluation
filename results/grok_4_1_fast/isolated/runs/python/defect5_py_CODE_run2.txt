The code fails on the line `new_list[len(new_list) - 1] = element` because `ArrayUtils._copy_list_grow1(array, inferred_type)` returns an empty list when `array` is None. This happens since `array` is Optional[List[T]] and can be None, in which case `_copy_list_grow1` likely creates a new list with length 0 to 'grow by 1'. Then `len(new_list) - 1` equals -1, causing an IndexError when trying to assign to a negative index.
