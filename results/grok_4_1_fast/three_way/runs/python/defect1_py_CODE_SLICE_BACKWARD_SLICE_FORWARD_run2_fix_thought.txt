The bug is in the negative hours_offset case where `minutes_offset = hours_in_minutes - minutes_offset` incorrectly subtracts the minutes component instead of adding it. For negative offsets, total minutes should be `hours_in_minutes + minutes_offset` (e.g., -60 + 30 = -30), not `-60 - 30 = -90`. Fix by always using addition: `total_minutes = hours_in_minutes + minutes_offset`.
