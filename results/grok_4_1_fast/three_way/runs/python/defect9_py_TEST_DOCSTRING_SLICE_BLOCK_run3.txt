The code fails the test_null_safety_inside_arrays_mixed test. In this test, StringUtils.replace_each("aba", ["a", "b"], ["c", None]) should return "cbc", replacing the first 'a' with "c", the second 'a' (now position 2) with "c", and the 'b' with None (empty string).
