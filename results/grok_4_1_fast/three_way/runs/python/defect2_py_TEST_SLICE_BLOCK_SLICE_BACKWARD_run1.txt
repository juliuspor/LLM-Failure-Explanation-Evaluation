The test suite fails because the `get_paint` method in `GrayPaintScale` raises a `ValueError` on lines 77-78 when the computed gray value `g` is outside [0, 255], instead of clamping it. Specifically:

The `int()` truncation does not clamp values to [0, 255]; it allows negative or >255 results, which the `if g < 0 or g > 255` check explicitly rejects instead of returning clamped RGB tuples like `(0,0,0)` or `(255,255,255)` as the tests expect.
