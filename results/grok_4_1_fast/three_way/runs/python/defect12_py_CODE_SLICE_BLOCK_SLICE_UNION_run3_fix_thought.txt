The bug occurs when lower > len(text). The code sets upper = len(text) if upper > len(text), but if upper < lower after this (len(text) < lower), it sets upper = lower, making upper > len(text). substring_java(0, upper) then raises IndexError due to end > len(s). Fix by clamping upper to min(upper, len(text)) after all adjustments, ensuring upper <= len(text) before substring calls.
