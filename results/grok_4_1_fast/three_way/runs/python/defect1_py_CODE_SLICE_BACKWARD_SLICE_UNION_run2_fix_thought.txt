The bug is in the incorrect handling of negative hour offsets. The line `minutes_offset = hours_in_minutes - minutes_offset` wrongly subtracts minutes_offset when hours_in_minutes is negative, producing incorrect large negative values (e.g., -60 - 30 = -90). It should always be `hours_in_minutes + minutes_offset` to correctly compute the total offset (e.g., -60 + 30 = -30). Remove the conditional subtraction and always add.
