Fix incorrect minutes_offset calculation for negative hours_offset. When hours_offset is negative, hours_in_minutes is negative, but the logic should add the absolute value of hours_in_minutes to minutes_offset (i.e., use + instead of -) to get the correct total minutes. Change 'minutes_offset = hours_in_minutes - minutes_offset' to 'minutes_offset = hours_in_minutes + minutes_offset' for the negative case.
