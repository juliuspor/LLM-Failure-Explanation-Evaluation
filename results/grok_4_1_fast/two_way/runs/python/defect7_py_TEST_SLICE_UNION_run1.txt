The code fails on line 761 in ClassUtils.to_class where it executes `array[i].upper()` for each element in the input array. When the array contains a None element (as in test_to_class_with_none_element with test_array = ["Test", None, 99.0]), accessing `array[i]` returns None when i=1. Calling `.upper()` on None triggers an AttributeError: 'NoneType' object has no attribute 'upper'. This line attempts to call upper() on every element regardless of its type, but None lacks this method, causing the crash before type(array[i]) is appended to classes on line 762.
