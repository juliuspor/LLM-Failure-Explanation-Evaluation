This loop runs for ALL indices in search_list/replacement_list (0 to search_length-1), with NO checks for None values. In test_null_safety_inside_arrays_mixed(), the inputs are search_list=["a", "b"] and replacement_list=["c", None]. When i=1, replacement_list[1] is None, so len(replacement_list[1]) fails.

Later search loops (lines 243+ and 285+) SKIP None entries with checks like 'replacement_list[i] is None', but this early buffer estimation loop has no such protection, causing the crash before any replacements occur.
