On line 185, the condition `if x == int(x)` fails because it assumes `x` can always be safely converted to an `int` using `int(x)`. This raises a `ValueError` if `x` is too large to fit in Python's integer range (e.g., exceeds approximately 9.22e18 on 64-bit systems).
