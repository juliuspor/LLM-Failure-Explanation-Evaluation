Calling `.upper()` on a non-string object like a class raises an `AttributeError: 'type' object has no attribute 'upper'`. This happens inside the for loop for each element `i` in `range(len(array))`. The line before assigns the result to `_` (discarding it), but the method call still executes and crashes. The subsequent `classes.append(type(array[i]))` never runs due to this error.
