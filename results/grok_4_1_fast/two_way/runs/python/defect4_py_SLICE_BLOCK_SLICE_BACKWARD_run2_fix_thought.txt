Fixed two bugs in _update_bounds: 1) Incorrect variable reference in max_middle_index block (self._min_middle_index instead of self._max_middle_index). 2) Missing 'else' clauses for all min/max blocks - when current index has smaller/larger value than existing extremum, the index was not updated because the assignment was outside the if block. Added proper else clauses to update indices only when conditions are met, preventing -1 values when valid extrema exist.
