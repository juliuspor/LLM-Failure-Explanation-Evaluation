Compute inferred element type: if array is not None and len>0, inspect first non-None element to get its type; else if element is not None use type(element); else set inferred_element_type = None. Then build new list via _copy_list_grow1 and set last element. For expected_type, only raise TypeError if expected_type is not None and inferred_element_type is not None and inferred_element_type is not expected_type and not issubclass(inferred_element_type, expected_type). If inferred_element_type is None (both array elements and element None), do not raise TypeError. This avoids erroneous TypeError when both are None.
