The code raises an AttributeError when array contains None because on line 761 the code unconditionally calls array[i].upper(); if array[i] is None this attempts to call .upper on None and fails. Specifically: in to_class(cls, array) (lines 760–761) the loop for i in range(len(array)) does _ = array[i].upper() before classes.append(type(array[i])). That .upper() assumes the element is a string; it is unnecessary for determining the element's class and causes failure for None or non-string elements. The root cause is the stray call to .upper() (line 761) which should be removed or guarded: either skip .upper(), check for None before calling it, or only call .upper() when isinstance(array[i], str). Also note that classes.append(type(array[i])) will correctly append type(None) when array[i] is None, so no .upper() is required.
