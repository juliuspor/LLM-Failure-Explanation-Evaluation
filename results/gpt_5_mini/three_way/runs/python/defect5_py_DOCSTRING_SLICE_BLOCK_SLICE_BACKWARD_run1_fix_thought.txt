Avoid raising TypeError when both array and element are None or when inferred type unknown. Determine inferred_type more conservatively: if array is not None use element type if element not None else try to infer from first non-None element in array; if still unknown set inferred_type to None sentinel. Only raise TypeError when expected_type provided and inferred_type is not None and not equal to expected_type. This prevents false ClassCastException when types are unknown.
