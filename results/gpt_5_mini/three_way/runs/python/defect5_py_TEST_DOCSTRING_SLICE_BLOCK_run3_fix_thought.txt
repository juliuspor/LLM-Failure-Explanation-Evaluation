When array is None and element is None but expected_type is provided, inferred_type was set to object causing a false cast error. Fix by preferring expected_type when array is None and element is None. Also refine cast check: only raise if inferred_type is not object and not equal to expected_type (i.e., real incompatibility). This allows unknown/object inferred types to be permissive or be set from expected_type when available.
