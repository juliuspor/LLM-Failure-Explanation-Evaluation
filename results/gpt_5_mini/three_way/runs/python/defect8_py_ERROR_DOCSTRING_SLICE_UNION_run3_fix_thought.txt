Allow empty country when locale string has two consecutive underscores (e.g. 'fr__POSIX'). Parse parts by underscores: 1 part -> language only; 2 parts -> language and country (country can be empty -> treated as ''); 3+ parts -> language, country (possibly empty), variant (remaining joined by '_'). Validate language is 2 lowercase letters; if country present and non-empty, validate 2 uppercase letters; variant can be any non-empty string. Raise ValueError for invalid formats.
