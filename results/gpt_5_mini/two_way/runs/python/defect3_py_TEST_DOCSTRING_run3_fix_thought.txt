Adjust index advancement in translate() so it increments by the number of UTF-16 code units consumed. When translate_codepoint returns 0 (no translation), compute codepoint at pos and use char_count() to advance by correct units. When translate_codepoint returns >0, advance by the sum of char_count of each consumed codepoint (using code_point_at as original). This ensures surrogate pairs are skipped fully and prevents stepping into low surrogates.
