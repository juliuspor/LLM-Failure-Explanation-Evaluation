Allow converting a generic object list to an expected_type list by verifying compatibility of each element. If array is None and element is None, fallback to object but validate against expected_type. Compute inferred_type from array if possible; otherwise from element; otherwise object. If expected_type provided and inferred_type is object, check every existing element (when array not None) and the new element to ensure isinstance(..., expected_type) (treat None as acceptable).
