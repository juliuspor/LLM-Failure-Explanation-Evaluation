Since check_val (0) equals value (which is also 0 if the original x was 0), the condition at line 192 is true, so mantissa //= 10 is executed, but 0 // 10 is still 0. Thus mantissa remains 0, and the loop continues forever because mantissa % 10 == 0 remains true.
