The bug is in the type inference logic. When `array` is not None, `type(array)` returns `list`, not the element type. We need to infer the element type from the first element of the array (if it exists) or default to `object`. When `array` is None, we should infer from `element` if possible, else default to `object`. Also, the condition `inferred_type == object` should compare the inferred element type, not the container type.
