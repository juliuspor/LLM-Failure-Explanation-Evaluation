On line 383, when the hours_offset is negative, the code incorrectly subtracts the minutes_offset from hours_in_minutes (which is the absolute value of hours_offset converted to minutes). This results in a smaller total minutes_offset than intended. The correct calculation should add the minutes_offset to the negative total, so it should be -hours_in_minutes + minutes_offset, or more simply, hours_in_minutes should be negated before adding minutes_offset.
