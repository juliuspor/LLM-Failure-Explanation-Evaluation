The code fails because the logic for handling negative hour offsets incorrectly subtracts the minutes_offset on line 383. When hours_offset is negative, the variable hours_in_minutes is negative (since it is hours_offset * 60). The code then does minutes_offset = hours_in_minutes - minutes_offset, which makes the total offset more negative than intended. For example, if hours_offset = -1 and minutes_offset = 30, hours_in_minutes = -60, and the calculation gives -60 - 30 = -90 minutes, which is -1.5 hours instead of the intended -1 hour and +30 minutes (which should be -30 minutes total). The correct operation should be addition: minutes_offset = hours_in_minutes + minutes_offset, as is correctly done for positive offsets on line 385. This error occurs specifically at line 383.
