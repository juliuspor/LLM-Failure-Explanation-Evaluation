The error occurs because the function is called with `text=None`. The docstring says if `text` is None, the function returns None. However, the function's implementation likely tries to call `len(text)` or perform a string operation on `text` before checking if it is None. The TypeError 'object of type 'NoneType' has no len()' indicates that `len()` was called on a None value. The fix is to move the `if text is None: return None` check to the very beginning of the function, before any operations that assume `text` is a string.
