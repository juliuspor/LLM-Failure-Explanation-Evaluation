The bug in `replaceEach` is that when `repeat` is true and `timeToLive` decreases, the method recursively calls itself but doesn't handle the case where `searchList` and `replacementList` might have been swapped due to memory optimization. The swapping of arrays (lines 3078-3084) changes the references, but the recursive call passes the original arrays, causing incorrect replacements. The fix is to pass the swapped arrays to the recursive call.
