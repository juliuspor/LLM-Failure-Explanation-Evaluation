# Documentation Index

This directory collects the stable documentation for the thesis repo.

## Sections

- `pipeline/` — experiment design, pipeline flow, and slicing process.
- `evaluation/` — evaluation methodology, prompts, and explanation-metrics analysis notes.
- `fixing/` — fix generation, comparison workflow, and Java no-explanation correctness notes.
- `reporting/` — report metric references, Halstead notes, and the tracked RQ4 output tables in `reporting/rq4_outputs/`.

## Reading Order

If you only need the essentials, start here:

1. `pipeline/pipeline_overview.md`
2. `evaluation/evaluation_and_llm_methodology.md`
3. `fixing/fix_process.md`
4. `reporting/report_metrics_reference.md`
